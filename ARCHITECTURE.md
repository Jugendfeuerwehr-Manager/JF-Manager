## Request Flow Examples

### 1. Frontend SPA Request
```
Browser вҶ’ http://localhost/
  вҶ“
Nginx: Serve /usr/share/nginx/html/index.html
  вҶ“
Browser loads Vue.js app
  вҶ“
Vue Router handles client-side routing
```

### 2. API Request
```
Browser вҶ’ http://localhost/api/v1/orders/
  вҶ“
Nginx: proxy_pass to http://backend:8000/api/v1/orders/
  вҶ“
Django REST Framework вҶ’ ViewSet вҶ’ Database
  вҶ“
JSON Response вҶҗ Django вҶҗ Nginx вҶҗ Browser
```

### 3. Django Admin Request
```
Browser вҶ’ http://localhost/admin/
  вҶ“
Nginx: proxy_pass to http://backend:8000/admin/
  вҶ“
Django Admin Interface вҶ’ Database
  вҶ“
HTML Response (with /static/ assets)
```

### 4. Static File Request
```
Browser вҶ’ http://localhost/static/admin/css/base.css
  вҶ“
Nginx: Serve from /static/ volume
  вҶ“
Response with Cache-Control: 30 days
```

### 5. Upload Request
```
Browser вҶ’ POST http://localhost/api/v1/attachments/
  вҶ“
Nginx: proxy_pass to backend:8000
  вҶ“
Django: Save file to /uploads/
  вҶ“
Response with file URL: /uploads/filename.pdf
```
